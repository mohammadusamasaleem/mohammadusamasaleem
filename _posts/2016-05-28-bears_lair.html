The website promotes verifiability of RNA-seq experiements. background: > This website is directly related to sleuth, another project I've contributed to. Sleuth is a differential analysis tool and is the first of such tools to be able to host its analysis live in a browser. If sleuth could turn RNA-seq analyses into webpages, the natural next step was to create a database that housed all of these analyses in a single place â€“ thus, the bear's lair was created.

The goal of the website is to have all authors of RNA-seq papers submit their raw data to the website, and then have our automated server process and upload their results. In doing so, individual authors increase their paper's credibility, and the bioinformatics community benefits from more transparency. Furthermore, by having the analysis live, users can continually discover new results that may have been missed in the original publication. my_role: > I designed the entirety of the website. The website is build using Jekyll and Bootstrap and hosted by Github Pages. The table in the analysis tab, which features the papers we analyzed, is powered by the jQuery DataTables plug-in.

I also designed the back-end automated workflow that analyzes RNA-seq data and uploads the results to our server. The workflow is made using Snakemake, a build-system created from python and frequenty used in computational biology. The workflow is depicted below: --- Reproducibility is critical to experiemental results. To this end, many RNA-seq papers archive their data in the Sequence Read Archive. This data is theoretically enough to reproduce the results of the original paper; doing so is typically infeasable due to the computational power required to process the results and the computational biology background needed to select the right computational tools.

In order to make RNA-seq results transparent, we (the Pachter Lab and I) created the bear's lair. It is a website that hosts interactive reanalyses of publicly available RNA-seq data. The analyses are automatically run by a back-end Snakemake buildsystem and performed by kallisto and sleuth. The website allows anyone to inspect the results of RNA-seq experiments without needing computing power or extensive knowledge of alignment/analysis tools.
